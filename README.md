# Parser Combinator

The easiest way to build robust parsers.

## XDebug

Turn off XDebug, as it will make things much slower. If you need it, increase the xdebug.max_nesting_level ini option:

```php
ini_set('xdebug.max_nesting_level', 5000);
```
